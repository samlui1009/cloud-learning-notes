# What IS Atlassian Bamboo?
<p>Establishing automated CI/CD pipelines is critical, especially in the world of DevOps. While many DevOps platforms (GitHub Actions, Jenkins) exist, the tool utilized in my role is <b>Atlassian Bamboo</b>.</p>

<p>Bamboo is a Java-based, platform-independent automation server, developed by Atlassian, utilized specifically for Continuous Integration and Continuous Deployment. It allows for the automatic build, integration, testing and deployment for software applications.</p>

<p>As a general whole, Bamboo is structured in the following way, using the concept of plans, jobs and tasks to configure and order the desired actions within the pipeline.</p>
